Document Tools

The Document Tools Plugin allows you to select a document and scale, crop and rotate in one action.

Basic Options

Documents
A the list of open documents to choose from.
Refresh
Refresh the list of documents.

Transform Options

The transform options.

Canvas Size

The cropped section.

Width
The width of the crop area.
Height
The height of the crop area.
Offset
The x and y coordinates of the crop.
