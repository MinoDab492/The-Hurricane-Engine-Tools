Assign Profile To Image

This allows you to assign a profile to an image instead of converting it to that profile.

The difference is that it allows only interpreting the colors by the new profile, but not change any of the values.

It can be found in Tools → Assign Profile To Image

It will present a list of profiles for the current image’s color model(As changing color model or bit depth will require conversion).

Press OK and the pixels will be interpreted differently.
